# npmコマンドをグローバルに利用できるようにalias設定

- ~/.zshrc に追加:
  ```bash
  alias note='npm --prefix ~/dev-notes run note'
  alias wrapup='npm --prefix ~/dev-notes run wrapup'
  ```

- 設定を反映: `source ~/.zshrc`

- どこからでも実行可能:
  ```bash
  note "Expo Routerで白画面を調査"
  wrapup "今日のまとめ"
  ```
